### Installation

1. Get a free API Key at [https://newsapi.org/](https://newsapi.org/)
2. Clone the repo
   ```sh
   git clone https://github.com/ThGnommy/react-native-news-app.git
   ```
3. Install YARN packages
   ```sh
   yarn
   ```
4. Create a .env file and insert
   ```js
   NEWS_APIKEY=YOUR_API
   ```
